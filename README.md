# ⚡ Shree Krishna Electricals

An Electrical Shop Management System built using Django.

## 🌐 Live Demo

https://electrical-shop-7pcp.onrender.com

## 💻 GitHub Repository

https://github.com/varunbhat678/electrical-shop

## 📌 Features

- Product Categories
- Product Details
- Product Images
- Responsive Bootstrap Design
- Django Admin Panel
- About Page
- Contact Page

## 🛠 Technologies Used

- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite
- Git
- GitHub
- Render

## 🚀 Installation

```bash
git clone https://github.com/varunbhat678/electrical-shop.git
cd electrical-shop
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
